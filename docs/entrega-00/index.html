Entrega 00 - Trabajos Representativos

En esta oportunidad se nos solicito mostrar tres trabajos distintos que hayamos realizado.

Templo Mexicano

Para “Modelacion 2D Y 3D” con una foto y algunas medidas de referencia debimos construir un render con el programa “Rhyno 7, 3D Models”. Decidi iniciar el modelado de este templo ubicado en MÉXICO con los pilares para posteriormente subir hasta tapar las cara posterior. En las siguientes imagenes se puede ver parte del proceso.

Bú Ho - Espantacucos LED para niños

Para “Taller de diseño de interfaces” se nos solicito construir un objeto “hechizo”, es decir, un objeto a partir de desechos, objetos rotos, basura o reciclaje. Ademas de esto habia una condición la cual era que el objeto debia incluir plasticina conductiva casera. Me costo mucho conformarme con alguna idea ya que sentia que la mayoria de estas eran muy simples, finalmente opte por hacer una lampara LED o espantacucos LED con forma de animal, especificamente con forma de BÚHO. Para hacer esto utilice un Bidón de 5 litros de agua vacio, pintura acrílica, tijeras, corta cartón, pinceles, dos luces led, plasticina conductiva y una batería Duracell de 9V. En las siguientes imagenes y videos podras ver todo el proceso paso por paso.

“111 Pyschedelic Salvador Charly” Digital RubeGoldberg Machine

Para “Desde el átomo al bit” se nos solicito realizar una “Rubegoldberg Machine” de manera digital con el programa de animacion “UNITY”. Además se nos pidio que la grabacion que mostrara el circuito contuviera 7 tomas distintas.
